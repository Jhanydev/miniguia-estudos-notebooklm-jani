## 🧪 Aplicação Prática no NotebookLM

Para garantir que este projeto não é apenas teórico, todas as interações foram realizadas no NotebookLM com fontes reais.

### 📌 Etapas realizadas

1. Criação de notebook com tema definido
2. Upload de fontes confiáveis (artigos e relatórios)
3. Testes com prompts exploratórios
4. Refinamento de prompts (engenharia de prompts)
5. Geração de material estruturado (resumo + glossário)

---

### 📸 Evidências

Exemplos de interações realizadas:

* Prompt inicial (genérico)
* Prompt refinado (mais específico)
* Prompt crítico (análise de riscos)

📂 Ver pasta `/assets` para prints

---

### 💡 Principais aprendizados práticos

* Prompts genéricos geram respostas superficiais
* Contexto melhora drasticamente a qualidade da resposta
* A IA responde melhor quando recebe instruções claras
* O pensamento crítico continua sendo essencial

---

### 🧠 Insight mais importante

> Saber perguntar é mais importante do que saber responder.
